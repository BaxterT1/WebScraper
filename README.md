This is a very basic image scraper for any html webpage. Just paste the URL when prompted of the webpage that you want to scrape. 
* Currently only scrapes the image in the resolution as its displayed on the webpage, meaning you can't get full resolution images.
* 
Future plans: 
_Scrape full resolution images._

*HOW TO RUN* 
1. Navigate to project directory in the terminal using /cd/Webscraper
2. Run api.js using 'node api.js'

NOTE: .env file is not included, to ensure the api call is successful create an api key on https://extract.pics/ and paste it in the .env file with the variable: API_KEY = *your key*.
